ADA

Learn ADA using Aurthor Vargas Lopes' AVLAda9x shareware compiler.

FORTRAN 90

FORTRAN 90 support files help you setup and use the Essential Lahey FORTRAN 90 (ELF90) Compiler from within Visual FlowCoder . 

HTML/VB SCRIPT

Allows VFC' parse manager system to flowcode imbedded VB Script code. The .key file also integrates Stephen Le Hunte's HTML Reference Library WinHelp with VFC's help button.

JAVA

Extra Java support files help you use Sun's Java JDK compiler , or Mircosoft's Visual J++, from within Visual FlowCoder . These key files also integrate Bill Bercik's Java WinHelp with VFC's help button letting you hilight a flowcoded java statement and click help to automatically search.

VHDL

VHDL support files help you setup and use Accolade Design Automation's Peak VHDL system from within Visual FlowCoder .
